# clickエフェクトコレクション
クリック／タップ発動型のビジュアルエフェクト集。

## デモ動画
https://github.com/zenn24ct/click-cutieeffect/assets/demo.mp4


## 概要
- JavaScript のみで実装（外部ライブラリ不使用）
- 画面クリック位置にエフェクトを生成
- メニューからエフェクトを切り替え可能

## 収録エフェクト
- インクの広がり
- 星の爆発
- 波紋
- 花火
- 炎
- 渦巻き

## 使い方
- `index.html` をブラウザで開く
- クリック／タップでエフェクト発動
- 左上メニューでエフェクトを切り替え
