# カラーインク・スプラッシュ（Color Ink Splash）

画面をタップ／クリックすると、カラフルなインクがポンッと広がるシンプルな表現デモです。  
短い JavaScript で動作し、見た目が綺麗で癒されるビジュアルを楽しめます。

---

## 特徴
- ランダムで綺麗な色のインクが広がる
- タップ／クリックに反応してスプラッシュ生成
- 10〜20行程度のシンプルな JavaScript
- 1 ファイル（HTML）だけで動作
- 残像・重力・透明フェードなど自然な動きを再現

---

## 使い方
1. `index.html` を作成
2. 提供されたコードを貼り付け
3. ブラウザで開くだけ  
   - PC：クリック  
   - スマホ：タップ

---

## コード構成（概要）
- `canvas` に描画
- タップ位置から複数のパーティクルを生成
- `hsl()` で色をランダム生成
- `color-mix()` とアルファ値で綺麗にフェード
- わずかな重力と残像で自然な表現

## カスタマイズ例
- インクの数やスピードを増やす
- 色の明るさ・彩度の調整
- 背景画像を設定して写真に色が付くようにする
- ブラシの種類（墨汁・爆発・散布）を追加
- スローモーション風の演出
